# 청소 당번 뽑기 (1–30)

간단한 웹 기반 프로그램입니다. `index.html`을 브라우저에서 열면 동작합니다.

기능
- 1번부터 30번까지 번호 표시
- 번호 클릭으로 수동 비활성화/활성화
- `랜덤 뽑기` 버튼으로 활성화된 번호 중 무작위 선택
- `뽑힌 번호 자동 비활성화` 체크 시 뽑힌 번호 자동 비활성화
- 뽑기 기록(최근 20개)

사용 방법
1. 파일들을 동일 폴더에 두고 `index.html`을 브라우저에서 엽니다.
2. 번호를 클릭해 제외할 번호를 수동으로 설정할 수 있습니다.
3. `랜덤 뽑기`를 눌러 결과를 확인합니다.
4. `초기화`는 모든 번호를 초기 상태로 되돌립니다.

간단히 실행 (Windows):

```powershell
Start-Process index.html
```
